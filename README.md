# COVIEW

회의 참여자의 화면과 비디오를 동시에 공유할 수 있는 1:1 온라인 화상 회의 플랫폼입니다.

## What I used

- Spring Boot
- JPA
- thymeleaf
- Spring Security
- WebRTC
- Bootstrap
